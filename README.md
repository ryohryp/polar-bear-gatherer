# 🧊 Polar Bear Gatherer  
**短時間サバイバル × 白熊バトル × 広告再現ブラウザゲーム**

---

## 🎮 概要
「Polar Bear Gatherer」は、極寒の地で資源を集め、  
焚き火で体温を維持しながら白熊を倒すことを目的とした  
**HTML5ブラウザ向けサバイバルゲーム**です。

広告でよく見た「資源を集めて進化するサバイバル」を  
短時間で遊べるよう再現し、**“触って気持ちいい”操作感**を最優先に設計しています。

---

## 🧱 技術構成

| 項目 | 内容 |
|------|------|
| **開発環境** | VS Code + Live Server + GitHub Pages |
| **技術** | HTML5 Canvas + JavaScript (ES6 Modules) + CSS3 |
| **描画** | Canvas 2D Context（960×540固定） |
| **入力** | キーボード / D-Pad / タップ起点ドラッグ（バーチャルスティック） |
| **デザイン** | 2Dピクセル調・高コントラスト |
| **サウンド** | 8bit効果音（OGG/WAV予定） |
| **将来的対応** | PWA（`manifest.webmanifest` + `service-worker.js`） |

## 🧍 プレイヤー仕様

| パラメータ | 内容 |
|-------------|------|
| `x, y` | 位置（2000×1400のワールド内） |
| `r` | 当たり判定（12px） |
| `sp` | 移動速度：2.0 |
| `hp` | 体力（0でゲームオーバー） |
| `cold` | 体温（0でHPが減少） |
| `atk` | 攻撃力（素手10／槍22） |
| `atkCD` | 攻撃クールダウン（18frame） |
| `hasSpear` | 槍クラフト済みフラグ |

---

## 🔥 焚き火（Fire）

| 項目 | 内容 |
|------|------|
| `heat` | 温度（0〜100） |
| `embers` | 残り火ゲージ（徐々に減少） |
| 効果 | 近距離(60px以内)で体温上昇・HP回復 |
| 自動補給 | `heat<70` または `embers<10` の時、木材を自動投入（CD:120frame） |

---

## 🌲 木と資源

| 項目 | 内容 |
|------|------|
| 木（trees[]） | 45本生成。HP=30で倒木化。32px以内で自動伐採。 |
| ドロップ（drops[]） | 木：木材 / 熊：肉。28px以内で自動拾得。 |
| インベントリ（inv） | 木材・肉の所持数を管理。木材10で槍クラフト可能。 |

---

## 🐻 白熊（Bear）

| 項目 | 内容 |
|------|------|
| HP | 150 |
| 攻撃力 | 8 |
| 追跡 | 槍クラフト後にaggro化しプレイヤーを追跡 |
| 攻撃条件 | 26px以内で攻撃（CD:35frame） |
| 撃破時 | 肉ドロップ・BEAR HPバー非表示 |

---

## ⚔️ アクション・クラフト

| アクション | 条件 | 効果 |
|-------------|------|------|
| 槍クラフト | 木材10以上・未装備 | 槍を装備し攻撃力UP、熊が出現・覚醒 |
| 焚き火投入 | 木材1消費・近距離 | 火力上昇（`heat+18`, `embers=60`） |
| 伐採 | 木近距離で自動伐採 |
| 拾得 | 落ちている資源を自動回収 |
| 攻撃 | 熊が近距離時に自動攻撃 |
| 再開（Rキー） | HP0時 | リスタート |

---

## 🧊 温度・HPシステム

| 状況 | 変化 |
|------|------|
| 焚き火近距離 | `cold += 0.25 + heat×0.004` |
| 遠距離 | `cold -= 0.04 + (100-heat)×0.0006` |
| HP回復 | 焚き火近距離かつHP<100で `+0.06/frame` |
| HP減少 | 体温0以下で `-0.15/frame` |

---

## 💀 ゲームオーバー

- HPが0になると、**全処理停止＋中央に “GAME OVER / Rで再開” 表示**  
- HUD更新は停止し、ログはフェードせず保持  
- Rキーで状態リセット → 再開  

---

## 📱 操作方法

### バーチャルスティック（スマホ／タブレット）
- **タップ起点からのドラッグ方向**で移動。  
- 離すと停止。  
- `DEADZONE=10px`, `MAX_RADIUS=72px`。  
- PCではマウスドラッグでも同様動作。

### キーボード／D-Pad
| 入力 | 動作 |
|------|------|
| WASD / ↑↓←→ | 移動 |
| C | 槍クラフト |
| R | 再スタート |
| D-Pad | モバイル用補助入力 |
